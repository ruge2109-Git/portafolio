Panel administrativo

Información del proyecto

  • Categoría: Desarrollo Web
  • Cliente: Freelance Workana
  • Tecnologías usadas: :

Esta aplicación fue creada para un cliente obtenido en Workana, consiste en un panel administrativo en el que el usuario sube uno o más archivos con una base de información, la cual se actualiza en la base de datos y genera por medio de distintas consultas SQL diferentes reportes y KPI's
