Esta aplicación fue creada como práctica de MEAN(Mongo, Express,Angular,Nodejs) , teniendo un diseño basado en una plantilla de Createx. En el backend se usa autenticación JWT para las solicitudes Api, se usa socket.io para manejar el carrito en tiempo real. El frontend se divide en dos proyectos, uno para el administrador y el otro para el cliente final. Ambos proyectos se conectan al mismo backend. Teniendo de esta manera un ecommerce bastante completo con todos los módulos necesarios, (Productos, categorías, carrito, pasarelas de pago, descuentos, códigos de promoción, etc)
